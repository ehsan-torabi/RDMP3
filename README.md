# RDMP3
RDMP3 is a simple tool for removing duplicate MP3 files. Provide the paths as command-line arguments (within double quotes) like this:
```
python Main.py ".../.../FOLDER 1" ".../.../FOLDER 2" ".../.../FOLDER 3" ".../.../FOLDER 4" ......
```
